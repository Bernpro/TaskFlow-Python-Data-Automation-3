# 📊 Sistema de Automatización de Reportes Ejecutivos en Python

Una aplicación inteligente desarrollada en Python y optimizada para entornos interactivos (como Google Colab y Jupyter) que transforma archivos de Excel planos y crudos en reportes financieros estilizados con diseño corporativo y fórmulas automatizadas.

---

## 🚀 Características Clave

* **Carga de Archivos Dinámica:** Permite al usuario subir cualquier archivo de Excel (.xlsx o .xls) de forma directa a la aplicación mediante componentes de interfaz nativos.
* **Procesamiento de Datos Logístico:** Utiliza la librería `pandas` para estructurar la matriz de datos, calcular impuestos (IVA 16%) y proyectar precios finales de forma automática.
* **Diseño Ejecutivo Avanzado:** Utiliza `openpyxl` para dar formato estético profesional (esquema de color azul corporativo, tipografía unificada, alineaciones ejecutivas y bordes limpios).
* **Fórmulas Nativas de Excel:** Inserta de manera automática renglones de resumen utilizando funciones reales de Excel (`=AVERAGE`), evitando datos estáticos y manteniendo el archivo dinámico para el usuario.
* **Auto-ajuste de Columnas:** Escanea el largo de los textos para evitar que la información se muestre recortada (`###`), garantizando una lectura inmediata.

---

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Manipulación de Datos:** `pandas`, `numpy`
* **Estilizado de Hojas de Cálculo:** `openpyxl`
* **Entorno de Ejecución:** Google Colab / Jupyter Notebooks
* Task bot: Selenium or el equivalente.  

---

## 📂 Instrucciones de Uso Cortas

1. Prepara un archivo Excel que contenga las columnas `Producto` y `Precio_Base`.
2. Ejecuta el script en Google Colab.
3. Haz clic en el botón **"Elegir archivos"** y selecciona tu archivo desde tu computadora.
4. Descarga el archivo resultante `Reporte_Automatizado_Final.xlsx` desde la barra de archivos lateral izquierda de Colab.

# 📊 Executive Report Automation System in Python

An intelligent application developed in Python and optimized for interactive environments (such as Google Colab and Jupyter) that transforms plain, raw Excel files into stylized financial reports with corporate design and automated formulas.

---

## 🚀 Key Features

* **Dynamic File Upload:** Allows the user to upload any Excel file (.xlsx or .xls) directly to the application using native interface components.

* **Logistics Data Processing:** Uses the `pandas` library to structure the data matrix, calculate taxes (16% VAT), and automatically project final prices.

* **Advanced Executive Design:** Uses `openpyxl` for professional formatting (corporate blue color scheme, unified typography, executive alignments, and clean borders).

* * **Native Excel Formulas:** Automatically inserts summary rows using real Excel functions (`=AVERAGE`), avoiding static data and keeping the file dynamic for the user.

* **Auto-Column Adjustment:** Scans the length of the text to prevent information from being truncated (`###`), ensuring immediate readability.

---

## 🛠️ Technologies Used

* **Language:** Python 3.x
* **Data Manipulation:** `pandas`, `numpy`
* **Spreadsheet Styling:** `openpyxl`
* **Runoff Environment:** Google Colab / Jupyter Notebooks
* Task bot:  Selenium or its equivalent.

---

## 📂 Short Instructions for Use

1. Prepare an Excel file containing the columns `Product` and `Base_Price`.

2. Run the script in Google Colab.

3. Click the **"Choose Files"** button and select your file from your computer.

4. Download the resulting file, `Reporte_Automatizado_Final.xlsx`, from the left-hand file bar in Colab.
